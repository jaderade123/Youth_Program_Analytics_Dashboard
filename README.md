# Youth Program Analytics Dashboard

## Project Overview

Built an end-to-end data collection and analysis system to track, analyze, and improve upon outreach, attendance, and CQI. I created this centralized system to track participant demographics, attendance, outreach effectiveness, and engagement strategies to inform recruitment strategies.

## Data Sources & Tools

### Data

- Program attendance records
- Form submissions 
- Recruitment pathways tracking
- Participant demographic data

### Tools

- Excel/Google Sheets
- Microsoft Forms

## Data Collection and Structure

1. Designed a centralized Excel workbook to consolidate all program data including attendance, demographic, engagement, and feedback.
2. Data was structured using tables and dropdowns where able to reduce inconsistencies and fast-track workflow.
3. Created a scalable system that allows for new events and participant entries without compromising formulas and data structure.

## Analysis and Methods

- Created Excel formulas for automated tracking:
  1. Total attendance for the YTD
  2. Counts of total youth interested
  3. Number of events attended
  4. Total repeat participation

- Created pivot tables to:
  1. Segment participants by demographic categories
  2. Analyze recruitment and engagement trends
  3. Compare outreach methods to attendance outcomes

- Transformed raw data into summary tables designed for ease of use for stakeholders and reporting for grant compliance
